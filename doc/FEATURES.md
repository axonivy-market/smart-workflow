# Smart Workflow Features

## Callable Smart Workflow Agent

The Smart Workflow Agent is available as a **callable subprocess** with signature:

```
Portal/SmartWorkflowAgent:invokeAgent(String,String,List<String>,Class)
```

> **Note:** This callable subprocess exists for Axon Ivy Portal features.
> For complete documentation on Smart Workflow interactions, refer to the Axon Ivy Portal documentation.
